# README - Knowledge Base

This directory contains example knowledge base documents organized by category.

## Structure

- `company/` - Company information, policies, and general information
- `products/` - Product documentation, features, and specifications
- `employees/` - Employee handbook, procedures, and guidelines
- `contracts/` - Contract templates and legal documents

## File Format

All documents should be in Markdown (.md) format for consistent processing.

## Adding Documents

1. Place documents in the appropriate category folder
2. Use descriptive filenames
3. Include relevant metadata in the document headers
4. Keep documents focused on specific topics for better retrieval

Replace these example files with your actual company knowledge base documents.
